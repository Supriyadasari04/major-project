# Emotion Aware AI Powered Mental Healtch Coach for Personalized Support

## Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/Supriyadasari04/major-project.git
   cd major-project
   ```

2. **Terminal 1**
   ```bash
   cd justly-emotion-service
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   uvicorn main:app --host 127.0.0.1 --port 8000 --reload
   ```

3. **Terminal 2**
   ```bash
   cd backend
   npm i
   npm run dev
   ```

4. **Terminal 3**
   ```bash
   npm i
   npm run dev
   ```
